#Elliptic Curves

This is a module on elliptic curves using python. 
For the moment more info on the website: http://oioannou.com/2015/research/

# Features

- Edwards curves
- Montgomery curves
- Encryption
- Re-encryption
- Decryption
- Proof of knowledge of re-encryption
- DDH verifier
